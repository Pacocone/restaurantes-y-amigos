# Restaurantes y Amigos

Una aplicación web para registrar y explorar visitas a restaurantes con amigos.

## 🌐 Aplicación en Vivo

La aplicación está desplegada en GitHub Pages:
**https://pacocone.github.io/restaurantes-y-amigos/**

## 🚀 Características

- **Registro de Visitas**: Añade visitas a restaurantes con detalles como fecha, ubicación, valoración y fotos de tickets
- **Exploración**: Descubre restaurantes por ciudad, país y usuario
- **Sistema de Amigos**: Conecta con otros usuarios y ve sus visitas
- **Multiidioma**: Soporte para español e inglés
- **Temas**: Modo claro y oscuro
- **Integración Firebase**: Autenticación de usuarios y almacenamiento de datos

## 🛠️ Tecnologías

- HTML5, CSS3, JavaScript (Vanilla)
- Firebase (Authentication, Firestore, Storage)
- GitHub Pages para el hosting

## 📱 Uso

1. Visita la aplicación en el enlace de arriba
2. Regístrate con tu email y contraseña
3. Comienza a añadir tus visitas a restaurantes
4. Explora restaurantes y conecta con amigos

## 🔧 Desarrollo Local

Para ejecutar localmente:

1. Clona el repositorio
2. Abre `index.html` en tu navegador
3. La aplicación se conectará automáticamente a Firebase
